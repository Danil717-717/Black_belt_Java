                 Annotation
    Аннотации - это специальные комментарии/
метки/метаданные, которые нужны для передачи
определенной информации.
   @Target показывает область кода, к которой
Аннотация может быть применена. Самые
распространенные области кода:
   * TYPE - class, interface, enum;
   * FIELD - поле класса;
   * METHOD - метод класса;
   * PARAMETER - параметры метода или конструктора;

   @Retention описывает жизненный цикл Аннотации
   * SOURCE - Аннотация видна в source коде,
              отбрасывается компилятором и уже в
              byte коде не видна
   * CLASS - Аннотация видна в byte коде,
             отбрасывается JVM во время выполнения
             программы;
   * RUNTIME - Аннотация видна во время выполнения
               программы
