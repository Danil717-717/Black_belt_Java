                enum(перечисления)
    enum - это способ ограничения определенного рода
информации конкретным списком возможных вариантов.
    Конструктор в enum имеет access modifier private
и не нуждается во внешнем вызове.
     enum является дочерним классом для java.lang.Enum
     Часто используемые методы: valueOf, values
     Можно создать в нутри класса, можно отдельным
классом.
